## ğŸ¨ Customization

### Colors
All colors are defined as CSS variables in `style.css`:

```css
--color-dark-bg: #0a0015;
--color-purple-dark: #1a0033;
--color-cyan: #00d9ff;
--color-magenta: #ff00ff;
--color-pink: #e600ff;
```

### Typography
The website uses the **Outfit** font family from Google Fonts. Change it in `style.css`:

```css
--font-primary: 'Outfit', sans-serif;
```

### Responsive Breakpoints
- **Desktop**: 1024px and above
- **Tablet**: 769px - 1024px
- **Mobile**: 768px and below
- **Small Mobile**: 480px and below
